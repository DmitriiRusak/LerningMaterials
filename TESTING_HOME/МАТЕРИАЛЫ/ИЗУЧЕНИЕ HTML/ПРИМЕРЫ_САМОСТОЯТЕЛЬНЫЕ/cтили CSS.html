стилизованный абзац


Этот абзац стилизован подругому - с помощью класса bold, но в этом классе не переписываются некоторые свойства описанные ранее для этого селектора(абзаца), а значит они всё-ещо применяются

Стилизованный абзац без применения класса а отсюда наченается тэг span который описан в стилях в связке с классом,таким образом получаем связку селектора с какимто классом ,но работать это будет только при применении данного селектора, и несмотря на то что этот класс описан в стилях самостоятельно и имеет свои свойства, в контексте к связке этого класса с селектором он преобретает другие, уникальные для связки именно с данным селектором свойства Здесь span заканчивается и действуют теже стили описанные для этого селектора.
